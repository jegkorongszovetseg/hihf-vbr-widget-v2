# Game Center

## Használat

```html
<mjsz-vbr-game locale="hu" game-id="73022" />
```

A mérkőzés azonosítót (gameId) meg lehet adni az URL-ben is paraméterként:
```bash
https://www.example.com/game?gameId=73020
```

## Példa

<ClientOnly>
  <mjsz-vbr-game
    locale="hu"
    game-id="73022"
  />
</ClientOnly>
