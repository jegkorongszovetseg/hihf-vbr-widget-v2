### Csapat változók:

| prop                 | Default | Leírás                        | Megjegyzés                       |
| :------------------- | :------ | :---------------------------- | :------------------------------- |
| team-filter-by-name  |         | Csak egy csapat mérkőzései    | pl.: `team-filter-by-name="UTE"` |
| externalTeamResolver |         | Csapat külső hivatkozása      |                                  |
| isTeamLinked         | false   | Csapat külső hivatkozás aktív |                                  |
