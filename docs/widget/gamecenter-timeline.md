# Game Center - Timeline

## Használat

```html
<mjsz-vbr-game-timeline locale="hu" game-id="77928" />
```

A mérkőzés azonosítót `game-id` meg lehet adni az URL-ben is paraméterként:

```http
https://www.example.com/game?gameId=73020
```
